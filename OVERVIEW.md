&#x20;                                                                               **AI Chatbot using Python**



**Project Overview:**



This project is a simple rule-based chatbot developed using Python. The chatbot responds to predefined user inputs and demonstrates fundamental Python programming concepts.



**Features:**



1.Interactive user conversation

2.Predefined responses

3.User-friendly command-line interface

4.Exit options using "bye", "exit", or "quit"



**Technologies Used:**



Python



**Concepts Used:**



1.Functions

2.Dictionaries

3.Loops

4.Conditional Statements

5.User Input Handling



**Sample Inputs:**



hi



how are you



what is python



who are you
